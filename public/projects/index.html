[4] Data Aggregation in Wireless Sensor Networks

Status: Completed

Duration: Nov 2009 - Dec 2009

Deliverables: Program Crossbow Technology’s MICA2 motes at photo-sensors in a network. The motes would send sensor readings packet to an aggregator mote located centrally at regular intervals. The aggregator mote will compute the difference in the readings of the two photo sensing motes and send a signal/message to a controller mote if the difference is greater than a preset threshold value. The controller mote would then take an appropriate action (sound the buzzer).

[3] Wireless Quiz System using Low Power Microcontrollers

Status: Completed

Duration: Oct 2009 - Dec 2009

Deliverables: Modify the presently existing Wired quiz buzzer system to a wireless buzzer system. The design includes 32 wireless buzzers which are to be networked and controlled by a central controller (CC). Each node (buzzer) will send an RF signal to the central controller along with its 5 bit ID. The CC will identify the node and will broadcast the ID indicating the team which pressed the buzzer first.

[1] Simons Game using Renesas QSK26A Board

Status : Completed

Duration : March 2009 - May 2009 (Spring ‘09 - ECGR5101 Embedded Systems Course)

Deliverables : A dueling Simon’s game is a dual player gameplay. The software efficiently manages game logic, user interface, and communication between devices. Features include alternating turns for pattern replication, LED and sound feedback (using PWM and a speaker), scoring, customizable settings, and error handling. Technical requirements involve compatibility with specified hardware (Renesas SKP26), real-time responsiveness, efficient memory usage, and the UART protocol to send and store scores to the connected PC.
